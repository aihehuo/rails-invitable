# RailsInvitable
Make the invitation function plugable.

## Usage
```ruby
# initializer
RailsInvitable.user_class = "User" # Or other user class name used in your project. Should be a string or symbol.
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rails_invitable'
```

And then execute:
```bash
$ bundle
```
